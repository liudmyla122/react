# Домашнее задание 18 - TypeScript

Этот проект содержит решения всех заданий по TypeScript.

## Структура проекта

- `src/task1.ts` - Задание 1: Объединение и пересечение типов
- `src/task2.ts` - Задание 2: Вложенные объекты и опциональные поля
- `src/task3.ts` - Задание 3: Интерфейс для функции calculateDiscount
- `src/task5a.ts` - Задание 5: Массив объектов и функции (Employee)
- `src/task5b.ts` - Задание 5: Наследование интерфейсов (Person, Student)
- `src/task6.ts` - Задание 6: Интерфейс для функции concatStrings
- `src/index.ts` - Главный файл для запуска всех заданий

## Установка зависимостей

```bash
npm install
```

## Запуск проекта

### Компиляция TypeScript
```bash
npm run build
```

### Запуск скомпилированного кода
```bash
npm start
```

### Запуск напрямую через ts-node
```bash
npm run dev
```
